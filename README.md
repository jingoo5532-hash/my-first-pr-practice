# My First PR Practice

This project provides a smple calculator with basic math operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Usage

```python
from calculator import add, subtract, multiply, divide

add(2, 3)       # 5
subtract(5, 3)  # 2
multiply(4, 3)  # 12
divide(10, 2)   # 5.0
```

## Tests

Run tests with:

```bash
pytest
```
